Object Detection

Ready-to-go solution for multiple object detection and classification

 

 About

This solution provides image analysis and detection of objects of various kinds.

Detection

The algorithm detects multiple objects in an image and gives coordinates to draw a bounding box around each object.

Classification

The algorithm suggests the most probable class along with its confidence level for each object.
